### Endpoints

#### {platform_id}/simulacion/tasasactuales

    <strong>get_tasasactuales</strong>
  </a> y retorna los resultados.
#### {platform_id}/simulacion/producto_tasas/{id_tasa_variable}

    <strong>get_variablerates_product</strong>
  </a> y retorna los resultados.
